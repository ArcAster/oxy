On the server:
oxy keygen > server_key   # This will create the host key
oxy keygen server_key     # This will print the public key of the host key

On the client:
oxy keygen > client_key   # Creates the key
oxy keygen client_key     # Prints the public key

On the server:
oxy server --peer=client_public_key -k "a symmetric key of your choice, or one will be generated and printed in the output"

On the client:
oxy client my.server.com -k "the same symmetric key" --peer=server_public_key

Press F10 to get to an interactive command prompt from the client. You can start portforwards and transfer files from this prompt. Try "help" at this prompt.

Press F12 to exit out of the client.
